# 美术示意图索引

生成时间：2026-04-30  
用途：用于项目讨论、玩法美术方向沟通、早期视觉参考。

这些图是根据玩法 GDD 和美术方向文档生成的概念示意，不是最终美术定稿。主要用来讨论方向、可读性和关卡空间表达。

## 1. 纸片剧场 / 立体书

![纸片剧场 / 立体书](./images/art_concepts/01_paper_theatre_pop_up_level.png)

对应方向：2D 纸片剧场 / 立体书。  
重点观察：纸片层级、舞台感、同一关卡在横版和俯视下的空间差异。

## 2. 工业遗迹 / 实验设施

![工业遗迹 / 实验设施](./images/art_concepts/02_industrial_experiment_facility.png)

对应方向：2D 工业遗迹 / 实验设施。  
重点观察：机关、管道、压力板、激光、机器人等元素是否适合承载玩法规则。

## 3. 手绘异空间

![手绘异空间](./images/art_concepts/03_handpainted_dream_space.png)

对应方向：2D 手绘异空间。  
重点观察：情绪、梦境感、记忆地图感，以及可交互物是否足够清楚。

## 4. 低多边形正交 2.5D

![低多边形正交 2.5D](./images/art_concepts/04_low_poly_orthographic_25d.png)

对应方向：3D 低多边形正交 2.5D。  
重点观察：同一 3D 场景从侧面和顶部观察时，玩法物体是否清楚。

## 5. 微缩模型 / 桌面沙盘

![微缩模型 / 桌面沙盘](./images/art_concepts/05_tabletop_diorama_puzzle_box.png)

对应方向：3D 微缩模型 / 桌面沙盘。  
重点观察：机关盒、模型感、实体道具和相机切换表现。

## 6. 超现实建筑 / 空间折叠

![超现实建筑 / 空间折叠](./images/art_concepts/06_surreal_folding_architecture.png)

对应方向：3D 超现实建筑 / 空间折叠。  
重点观察：概念感、空间变化表现，以及玩法可读性风险。

## 7. 推箱多用途机制示意

![推箱多用途机制示意](./images/art_concepts/07_mechanic_concept_push_box_multiuse.png)

对应方向：玩法机制示意。  
重点观察：同一个箱子在不同视角下作为平台、填路、阻挡、伤害道具的表达是否清楚。

## 8. 低多边形三渲二像素风

![低多边形三渲二像素风](./images/art_concepts/08_lowpoly_2d_pixel_render_style.png)

对应方向：低多边形 3D 造型 + 三渲二 + 像素风处理。  
重点观察：3D 空间感、像素化边缘、玩法物体可读性，以及这种风格是否适合横版 / 俯视切换。

## 9. 参考图风格延展：低多边形三渲二像素风

![参考图风格延展：低多边形三渲二像素风](./images/art_concepts/09_t3ssel8r_like_lowpoly_pixel_puzzle_level.png)

对应方向：基于参考图的低多边形、三渲二、像素化边缘、柔和遗迹草地风格。  
重点观察：这种更接近参考图的风格，是否适合承载推箱、平台、机关、断桥和俯视 / 横版切换关卡。

## 10. 实机感探索关卡

![实机感探索关卡](./images/art_concepts/10_ingame_style_exploration_ruins.png)

对应方向：接近实机截图的遗迹探索场景。  
重点观察：固定斜俯视相机下，树、柱子、石块、平台和道路是否有足够清晰的玩法边界。

## 11. 实机感推箱机关

重点观察：箱子、压力板、断桥、坑洞这些核心机制是否能在该风格下清楚表达。

## 12. 实机感视角切换布局

![实机感视角切换布局](./images/art_concepts/12_ingame_style_view_switch_layout.png)

重点观察：同一关卡里，平台逻辑和平面路线是否能被玩家直观看懂。

## 13. 实机感追逐 / 敌人压力

![实机感追逐 / 敌人压力](./images/art_concepts/13_ingame_style_chase_pressure.png)
